Compatibility

JDK8 Gradle 2.12 Tomcat 7

Installation

Configuration

You have to update src/main/resources/application.yml and specify RabbitMQ connection properties and the name of the exchange point, which should be created manually.

Building the library

$ ./gradlew clean assemble

Executing

Put the library (build/libs/remrem-publish-${VERSION}.war) to the $CATALINA_HOME/webapp and restart your Tomcat7.

Usage

The library supports POST requests
